# 🤖 GitHub定时提醒系统

## 📋 系统功能

✅ **完全自动化** - 每天11:00自动运行，无需人工干预  
✅ **Telegram提醒** - 到期自动发送消息到手机  
✅ **网页管理** - 随时随地添加/编辑提醒  
✅ **云端存储** - 所有数据保存在GitHub  

## 🚀 快速开始

### 1. 配置Telegram Bot（必需）

```bash
1. 打开Telegram，搜索 @BotFather
2. 发送 /newbot
3. 输入机器人名称（如：我的提醒机器人）
4. 输入用户名（必须以_bot结尾，如：my_reminder_bot）
5. 复制得到的Token（格式：1234567890:ABCdefGHI...）
